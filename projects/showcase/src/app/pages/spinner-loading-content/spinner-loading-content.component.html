Spinner / Loading

Herramientas para mostrar al usuario que se está cargando algún elemento.

Loading Screen

Presiona el botón y verás una pantalla sobrepuesta que bloquea la página para que el usuario no ejecute acción hasta que termine un determinado proceso.

Abrir loading-screen @if (showLoadingScreen()) { }

Valores por defecto:

  • [mostrarTexto]: 'Cargando datos...'
  • [textClassColor]: 'text-white'
  • [borderClassColor]: 'border-gray-300'
  • [spinnerClassColor]: 'border-t-blue-500'

[mostrarTexto] recibe un texto para mostrar en la pantalla de carga.
[textClassColor] debería recibir una clase de tailwind relacionada con el color del texto.
[borderClassColor] debería recibir una clase de tailwind relacionada con el color del spinner de fondo.
[spinnerClassColor] debería recibir una clase de tailwind relacionada con el color del spinner que más destaque.
