Fonts

Se encuentran indicados en la biblioteca los fonts que deben ser descargados por el usuario en su proyecto. Estos se pueden ver en "peerDependencies" del package.json de ngx-cygnus-ui.

Instalación

Se deben instalar las fuentes que se requieran con npm 

          
            npm install @fontsource/dm-sans
            npm install @fontsource/inter
            npm install @fontsource/nunito-sans

O bien agregarlas directamente al package.json del proyecto y ejecutar npm i

En styles.css de tu proyecto, debes importar las fuentes y ahí mismo puedes crear variables y clases según sea necesario.
