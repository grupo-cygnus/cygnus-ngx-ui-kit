Cygnus Javi Table

Tablas de datos con modo híbrido (Tabla/Tarjetas), filtrado inteligente y persistencia de vistas favoritas.

Original: https://github.com/Javibluebell/vanilla-js-data-table-component

Contenido

Este componente se compone de la siguiente rama de archivos: 

          
            cygnus-javi-table
            ---cygnus-javi-table.component.html
            ---cygnus-javi-table.component.ts
            ---data-table.models.ts
            ---data-table.service.ts
            ---format-value.pipe.ts
  • [cygnus-javi-table.component.html]: Plantilla de la tabla dinámica.
  • [cygnus-javi-table.component.ts]: Typescript general de la tabla.
  • [data-table.models] : Interfaces para estructurar elementos que se muestran en la tabla.
  • [data-table.service]: Servicio para gestionar el almacenamiento en storage.
  • [format-value.pipe] : Pipe propio para fechas y números.

Instalación requerida

Para poder exportar documentos en Excel, se requiere la instalación y configuración de SheetJS en el proyecto donde se utilizará la biblioteca. Si esto no se realiza ni se configura, se exportarán documentos .csv

Instalación: 

          npm install xlsx

Configuración del archivo angular.json: 

          
            "architect": {
              "build": {
                "options": {
                  "scripts": [
                    "node_modules/xlsx/dist/xlsx.full.min.js"
                  ]
                }
              }
            }

Después de este cambio, es obligatorio reiniciar el servidor de desarrollo (ng serve) para que Angular incluya el script en el bundle.

Configuración

Añadir un array con los datos que se requiere mostrar.

Añadir un objeto del tipo DataTableConfig (interface encontrada en data-table.models) con el detalle del funcionamiento de la tabla. Observar que el arreglo creado en el paso anterior va enlazado en data.

En el documento .html añadir la plantilla con la configuración realizada.
